Step 1: Tạo thư mục con trong thư mục lớn (tên gì cũng được)
Step 2: Mở terminal chạy "cd <tên thư mục con>" rồi chạy "npx create-react-app ." để tạo react (nếu đã có react có thể skip 
        tới step 3 sau khi trỏ tới thư mục con)
Step 3: Chạy lệnh trong terminal "npm install @mui/material @emotion/react @emotion/styled" để cài thư viện Material-UI
        Tiếp chạy "npm install recharts"
                  "npm install react-router-dom"
                  "npm install react-hook-form"
Step 4: Chạy lệnh "npm start" sau khi lên trang web "http://localhost:3000/" sau đuôi '/' thì nhập: movies, rooms, schedules,...
